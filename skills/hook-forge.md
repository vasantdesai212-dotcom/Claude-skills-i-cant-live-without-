# Hook Forge

A skill for generating high-converting content hooks using proven psychological angles.

## What It Does

Hook Forge takes any content idea, topic, or draft and produces multiple hook variations
using different psychological frameworks. Each hook is designed to stop the scroll and
compel the audience to keep reading or watching.

In the game of content, the hook is everything. A great hook can 2–5x the performance
of the exact same content with a weak opener.

## How to Use

1. Type `/hookforge` in Claude
2. Paste your content, topic, or idea
3. Receive 5–10 hook options across different psychological angles

## Psychological Angles Used

| Angle | Example |
|-------|---------|
| **Curiosity** | "There's a folder on GitHub that quietly replaces $200/month worth of subscriptions." |
| **Contrast** | "Most people think saving money on software means switching to cheaper plans." |
| **Controversy** | "Paying for MidJourney in 2026 is a skill issue, not a budget decision." |
| **Social Proof** | "The top 1% of developers I know all share the same secret stack — and every tool is free on GitHub." |
| **Fear of Missing Out** | "Everyone building AI apps in 2026 knows about this GitHub repo. Do you?" |
| **Empathy** | "If you've ever felt overwhelmed by SaaS bills, this is for you." |

## Example

> **User:** `/hookforge` — Five GitHub repos everyone must try
>
> **Hook Forge Output:**
> - *Curiosity:* "There's a folder on GitHub quietly replacing $200/month of subscriptions."
> - *Contrast:* "Most people think saving money on software means switching to cheaper plans."
> - *Controversy:* "Paying for MidJourney in 2026 is a skill issue, not a budget decision."
> - *Social Proof:* "The top 1% of developers share the same free stack — here it is."

- Paste your full content for more tailored hooks

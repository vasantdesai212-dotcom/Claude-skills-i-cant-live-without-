# Infographic Builder

A skill for creating beautiful, interactive infographics on any topic — ready for
LinkedIn, Instagram, presentations, or educational content.

## What It Does

Infographic Builder takes any topic, concept, or dataset and generates a fully
interactive HTML infographic with:
- Professional layout and typography
- Hover animations and motion effects
- Scrollable sections for dense content
- Platform-appropriate dimensions (Instagram feed, LinkedIn, A4, etc.)
- Multiple visual styles to match your brand

## How to Use

1. Type `/infographic-builder` in Claude
2. Describe the topic or content you want visualized
3. Choose your platform/size (Instagram Feed, LinkedIn, Presentation, etc.)
4. Choose your visual style (Bold Pop, Minimal, Dark, Corporate, etc.)
5. Claude generates a fully interactive infographic

## Example

> **User:** `/infographic-builder` — Numerous psychological tactics brands use to
> make you buy things. Show it visually for Instagram feed. Bold Pop style. Interactive with motion.
>
> **Output:** A scrollable, hover-animated infographic showing 8–12 psychological
> persuasion tactics with icons, short descriptions, and smooth micro-interactions.

## Platform Presets

| Platform | Dimensions | Best For |
|----------|-----------|----------|
| Instagram Feed | 1080×1080px | Brand posts, carousels |
| Instagram Story | 1080×1920px | Swipe-up content |
| LinkedIn Post | 1200×627px | Thought leadership |
| LinkedIn Carousel | 1080×1080px | Slide-by-slide education |
| Presentation Slide | 1920×1080px | Talks, webinars |
| A4 Document | 794×1123px | Reports, handouts |

## Visual Styles

- **Bold Pop** — High contrast, vibrant colors, strong typography
- **Minimal** — Clean white space, muted tones, elegant layout
- **Dark Mode** — Deep backgrounds, neon accents, tech aesthetic
- **Corporate** — Professional blues and grays, grid-based
- **Playful** — Rounded shapes, bright palette, friendly feel

## Interactive Features

- Hover effects on each data point or section
- Scroll-triggered animations
- Expandable detail sections
- Tooltip overlays with extra context
- Animated counters and progress bars

## Output

- Self-contained interactive HTML file
- PNG/JPG export option for static platforms
- Responsive and embeddable

## Tips

- Give a list of points or data if you have one — Claude will format and visualize it
- Mention your brand colors for consistent styling
- Ask for a "carousel version" to split into multiple slides
- Works great for: stats, how-to guides, comparisons, timelines, frameworks, ranked lists
